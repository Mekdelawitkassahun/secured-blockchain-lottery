# Decentralized Lottery

A simple, transparent lottery system on the Ethereum blockchain where anyone can buy tickets and win prizes.

## Features

- 30-Second Rounds - Quick lottery rounds for fast results
- Multiple Ticket Types - Standard (0.01 ETH), Premium (0.05 ETH), VIP (0.1 ETH)
- Referral System - Earn 5 percent when friends use your referral code
- Provably Fair - Winner is randomly selected on-chain
- No Middleman - Prize pool goes directly to winners

## Technology Stack

- Smart Contract: Solidity 0.8.19
- Frontend: Next.js with TypeScript
- Web3: ethers.js
- Styling: Tailwind CSS
- Network: Sepolia Testnet
